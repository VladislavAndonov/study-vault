# Study Vault
A personal collection of study notes, concept explanations, and useful resources I gather while learning.
Topics include programming, computer science, math, and anything else I explore.

---

## Structure
- **notes/** – Longer lecture notes and topic breakdowns
- **concepts/** – Short explanations of specific ideas
- **snippets/** – Small code snippets and examples
- **resources/** – Links to books, articles, videos, references

---

## Purpose
This repository acts as my personal knowledge base and daily learning habit.  
It helps me:
- Stay consistent with learning
- Organize what I study
- Quickly revisit core concepts
